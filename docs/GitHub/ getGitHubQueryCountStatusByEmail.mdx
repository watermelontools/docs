# Add Email to GitHub Query Count Table

## Route

```
api/github/addEmailToGitHubQueryCountTable
```

## Method

POST

## Params

```
{
    "email": "string",
}
```

Where:

- email: email of the user whos query count is to be read

## Response

### Errors

If nothing comes in the `email` param

```
{
    "error": "no email"
}
```

### Success

```
[]
```
